About This Project

This project involves developing a web application for event registration and payment collection for an Event. The application allows users to submit team and member information, calculate total payment based on team size, and complete payments using QR code and UPI app integration. The tech stack includes HTML, CSS, and JavaScript for the front-end, while PHP and AJAX are used for handling form submissions and server-side processing.
